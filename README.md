# AI-Agent-Personas
Master Code Generation Instructions

## Code Generation
The babylon-toolkit folder contains the instructions the agent must use for generating code.

- [classic.md](babylon-toolkit/classic.md) are instructions using the standard html, css and javascript web application development
- [node-umd.md](babylon-toolkit/node-umd.md) are instructions using node.js with the UMD (Universal Module Definition) framework
- [node-esm.md](babylon-toolkit/node-esm.md) are instructions using node.js with the ES6 (ECMAScript Modules Level 6+) framework
